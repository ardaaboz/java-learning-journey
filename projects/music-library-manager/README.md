# Music Library Manager

Create a program that manages a collection of songs using the Java Collections Framework:

## Requirements

- **Create classes** for `Song` and `Album`.
- **Use an appropriate collection** to store songs in an album (no duplicates allowed).
- **Use a `Map`** to associate album titles with their respective `Album` objects.

## Implement Methods To:

- Add a new album
- Add songs to an album
- Find all songs by a specific artist
- List all albums sorted by release year
- Generate a playlist from multiple albums
